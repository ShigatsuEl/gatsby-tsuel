문자열을 뒤집는 함수를 작성하세요. 입력값으로 받는 **s**는 문자를 담은 리스트입니다.

`Example 1:`
```
Input: s = ["h","e","l","l","o"]
Output: ["o","l","l","e","h"]
```

`Example 2:`
```
Input: s = ["H","a","n","n","a","h"]
Output: ["h","a","n","n","a","H"]
```

`Constraints:`
```
1 <= s.length <= 105
s[i] is a printable ascii character.
```

## 알고리즘 설계

1. 리스트가 주어져 있으므로 리스트의 길이를 이용하도록 합니다.

2. 왼쪽 끝에서 바꿀 대상의 리스트 위치를 `left` 오른쪽 끝에서 바꿀 대상의 리스트 위치를 `right`로 지정합니다.

3. left가 right보다 커지기 전까지 left와 right에 위치한 리스트의 값을 서로 스위치합니다.

## 작성한 코드

```python{numberLines: true}
from typing import List


class Solution:
    def reverseString(self, s: List[str]) -> None:
        left, right = 0, len(s) - 1
        while left < right:
            s[left], s[right] = s[right], s[left]
            left += 1
            right -= 1
```

## 다른 방법

파이썬의 내장 메서드 reverser()를 사용해 간단하게 문자열을 뒤집을 수 있습니다.

```python{numberLines: true}
from typing import List


class Solution:
    def reverseString(self, s: List[str]) -> None:
        s.reverse()
```
