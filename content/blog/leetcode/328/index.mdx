짝수 노드 이후에 홀수 노드가 오도록 재구성 하세요.

공간 복잡도 **O(1)**, 시간 복잡도 **O(N)**을 만족해야 합니다.

`Example 1:`
```
Input: head = [1,2,3,4,5]
Output: [1,3,5,2,4]
```

`Example 2:`
```
Input: head = [2,1,3,5,6,4,7]
Output: [2,3,6,7,1,5,4]
```

`Constraints:`
```
n == number of nodes in the linked list
0 <= n <= 104
-106 <= Node.val <= 106
```

## 알고리즘 설계

1. 홀수 노드의 다음은 홀수가 가리키도록, 짝수 노드의 다음은 짝수가 가리킬 수 있도록 순환문을 반복합니다.

2. 순환문이 끝나면 홀수 연결리스트 다음에 짝수 연결리스트가 연결되도록 합니다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def oddEvenList(self, head: ListNode) -> ListNode:
        if head is None:
            return None

        odd = head
        even = head.next
        even_head = head.next

        while even and even.next:
            odd.next, even.next = odd.next.next, even.next.next
            odd, even = odd.next, even.next
        odd.next = even_head
        return head
```

even_head 값을 가지는 이유는 13번 라인에서 홀수 연결리스트를 짝수 연결 리스트와 연결하기 위해서 입니다.
