문자열 s가 주어졌을때 반복 문자가 없는 가장 긴 부분 문자열을 찾으세요.

`Example 1:`
```
Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with the length of 3.
```

`Example 2:`
```
Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.
```

`Example 3:`
```
Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.
```

`Constraints:`
```
0 <= s.length <= 5 * 104
s consists of English letters, digits, symbols and spaces.
```

## 알고리즘 설계

1. 이 문제는 문자열이 반복되는지 아닌지를 확인할 수 있는 방법을 구현해야 합니다.

2. 문자열을 순환하면서 처음보는 문자열이면 `array`리스트에 추가하고 그렇지 않다면 `array`리스트에서 중복 문자열을 제거해야 합니다.

3. 중복 문자열이 없는 문자열의 최대 길이를 max 메서드를 사용해 구합니다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        array = []
        result = 0
        for index in range(len(s)):
            if s[index] not in array:
                array.append(s[index])
                result = max(len(array), result)
            else:
                result = max(len(array), result)
                array = array[array.index(s[index]) + 1:] + [s[index]]
        return result
```
