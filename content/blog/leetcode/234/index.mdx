연결 리스트의 HEAD 노드가 입력으로 주어졌을 때, 입력값이 팰린드롬이면 True를 반환하세요.

`Example 1:`
```
Input: head = [1,2,2,1]
Output: true
```

`Example 2:`
```
Input: head = [1,2]
Output: false
```

`Constraints:`
```
The number of nodes in the list is in the range [1, 105].
0 <= Node.val <= 9
```

## 알고리즘 설계

1. 팰린드롬을 확인하기에 가장 좋은 자료구조가 있는데 앞뒤로 삽입, 제거가 가능한 **Deque**를 사용하도록 합니다.

2. 주어진 입력을 Deque에 삽입하고 pop() 결과값과 popleft()결과값이 다르다면 팰린드롬이 아닌 것이고 모두 같다면 팰린드롬으로 판정합니다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        deque: Deque = collections.deque()

        if not head:
            return True

        while head is not None:
            deque.append(head.val)
            head = head.next
            
        while len(deque) > 1:
            if deque.popleft() != deque.pop():
                return False
        return True
```

## 다른 방법

파이썬 내장 자료구조 리스트를 이용한 방법입니다. 하지만 Deque를 사용한 방법보다는 느리다.

```python{numberLines: true}
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        q: List = []

        if not head:
            return True

        node = head
        # 리스트 변환
        while node is not None:
            q.append(node.val)
            node = node.next

        # 팰린드롬 판별
        while len(q) > 1:
            if q.pop(0) != q.pop():
                return False

        return True
```
