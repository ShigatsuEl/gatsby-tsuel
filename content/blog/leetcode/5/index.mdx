가장 긴 팰린드롬 문자열을 출력하세요.

`Example 1:`
```
Input: s = "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```

`Example 2:`
```
Input: s = "cbbd"
Output: "bb"
```

`Example 3:`
```
Input: s = "a"
Output: "a"
```

`Example 4:`
```
Input: s = "ac"
Output: "a"
```

`Constraints:`
```
1 <= s.length <= 1000
s consist of only digits and English letters (lower-case and/or upper-case),
```

## 알고리즘 설계

홀수로 끝나는 팰린드롬과 짝수로 끝나는 팰린드롬을 나누어서 생각해보아야 합니다.

1. 문자열의 길이가 2보다 작거나 뒤집었을 때 서록 같다면 볼 것 없이 팰린드롬이므로 그대로 리턴하도록 합니다.

2. 순환문에서 가지는 인덱스 번호를 팰린드롬의 시작으로 잡고 홀수 팰린드롬과 짝수 팰린드롬의 최대길이를 구합니다.

3. 현재까지 가지고 있는 결과값과 홀수 팰린드롬, 짝수 팰린드롬의 길이를 서로 비교해 가장 큰 길이를 결과값으로 저장합니다.

4. 순환문이 끝날때까지 반복하고 결과를 반환합니다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def longestPalindrome(self, s: str) -> str:
        def helper(left:int, right:int) -> str:
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return  s[left + 1:right]

        if len(s) < 2 or s == s[::-1]:
            return s
        result = ''
        for i in range(len(s)):
            result = max(result, helper(i,i), helper(i, i + 1), key=len)
        return result
```
