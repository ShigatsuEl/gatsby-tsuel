정수로 이루어진 두개의 연결 리스트가 입력으로 주어졌을 때, 두 수의 합을 구해 연결 리스트로 리턴하세요.

`Example 1:`
```
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807.
```

`Example 2:`
```
Input: l1 = [0], l2 = [0]
Output: [0]
```

`Example 3:`
```
Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
```

`Constraints:`
```
The number of nodes in each linked list is in the range [1, 100].
0 <= Node.val <= 9
It is guaranteed that the list represents a number that does not have leading zeros.
```

## 알고리즘 설계

1. 가장 단순한 방법으로 연결리스트를 숫자로 바꾸어 더한 후 다시 연결 리스트로 바꾸어 반환하는 것이다.

2. 주의할 점은 반환할 값을 역순 리스트로 만들어야 한다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        n1 = ''
        n2 = ''

        while l1.next != None:
            n1 += str(l1.val)
            l1 = l1.next
        n1 += str(l1.val)
        while l2.next != None:
            n2 += str(l2.val)
            l2 = l2.next
        n2 += str(l2.val)

        sum = int(''.join(reversed(n1))) + int(''.join(reversed(n2)))
        sum = str(sum)
        
        next = None
        
        for val in sum:
            node = ListNode(val)
            node.next = next
            next = node
        return node
```
