배열 nums를 입력받아 합으로 0을 만들 수 있는 [nums[i], nums[j], nums[k]의 모든 집합을 출력하세요.

`Example 1:`
```
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
```

`Example 2:`
```
Input: nums = []
Output: []
```

`Example 3:`
```
Input: nums = [0]
Output: []
```

`Constraints:`
```
0 <= nums.length <= 3000
-105 <= nums[i] <= 105
```

## 알고리즘 설계

투 포인터를 이용해 문제를 풀어봅시다.

1. 배열에서 사용할 세 수 중에서 하나의 수를 순환문의 인덱스로 잡은 후 left와 right를 인덱스의 오른쪽과 배열의 마지막으로 잡는다.

2. nums[i] + nums[left] + nums[right]를 더한 sum 값이 0을 초과하면 right 포인터를 감소시키고 0 미만이면 left 포인터를 증가시킨다.

3. sum이 0이 되는 경우를 result에 추가하고 나머지 순환문도 똑같이 진행한다.

## 작성한 코드

```python{numberLines: true}
from typing import List


class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        result = []
        nums.sort()
        
        for i in range(len(nums) - 2):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
                
            left, right = i + 1, len(nums) - 1
            while left < right:
                sum = nums[i] + nums[left] + nums[right]
                if sum > 0:
                    right -= 1
                elif sum < 0:
                    left += 1
                else:
                    result.append([nums[i], nums[left], nums[right]])
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    left += 1
                    right -= 1
        return result
```
