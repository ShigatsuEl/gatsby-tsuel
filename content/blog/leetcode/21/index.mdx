정렬되어 있는 두 연결 리스트를 합치세요.

`Example 1:`
```
Input: l1 = [1,2,4], l2 = [1,3,4]
Output: [1,1,2,3,4,4]
```

`Example 2:`
```
Input: l1 = [], l2 = []
Output: []
```

`Example 3:`
```
Input: l1 = [], l2 = [0]
Output: [0]
```

`Constraints:`
```
The number of nodes in both lists is in the range [0, 50].
-100 <= Node.val <= 100
Both l1 and l2 are sorted in non-decreasing order.
```

## 알고리즘 설계

1. 두 연결리스트가 정렬되어 있다는 점을 이용합니다.

2. l1과 l2의 값을 비교해 작은 값이 왼쪽에 오게 하고 next는 그 다음 값이 엮이게 하도록 재귀 호출을 한다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
        if (not l1) or (l2 and l1.val > l2.val):
            l1, l2 = l2, l1
        if l1:
            l1.next = self.mergeTwoLists(l1.next, l2)
        return l1
```
