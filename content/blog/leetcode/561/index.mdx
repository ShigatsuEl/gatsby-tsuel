리스트 nums가 주어졌을때 min(a,b)의 합으로 만들 수 있는 가장 큰 수를 출력하세요.

`Example 1:`
```
Input: nums = [1,4,3,2]
Output: 4
Explanation: All possible pairings (ignoring the ordering of elements) are:
1. (1, 4), (2, 3) -> min(1, 4) + min(2, 3) = 1 + 2 = 3
2. (1, 3), (2, 4) -> min(1, 3) + min(2, 4) = 1 + 2 = 3
3. (1, 2), (3, 4) -> min(1, 2) + min(3, 4) = 1 + 3 = 4
So the maximum possible sum is 4.
```

`Constraints:`
```
1 <= n <= 104
nums.length == 2 * n
-104 <= nums[i] <= 104
```

## 알고리즘 설계

1. min(a,b)의 합이 어떻게 해야 가장 커질 수 있을까? 그 답은 가장 큰 수끼리 페어를 이루고 그 다음 가장 큰 수끼리 페어를 이루는 
형식으로 반복되는 것이며 이것은 nums 리스트가 정렬이 되어있는 상태라면 굉장히 쉽게 할 수 있다.

2. 오름차순, 내림차순 상관없이 정렬되어 있다면 순서대로 2개씩 페어를 이루어 min()값을 구해 더하면 가장 큰 값이 나오게 된다.

## 작성한 코드

```python{numberLines: true}
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        pair = []
        sum = 0
        nums.sort()
        for num in nums:
            pair.append(num)
            if len(pair) == 2:
                sum += min(pair)
                pair = []
        return sum
```

## 다른 방법

위에서 정렬한 것까지는 똑같으며 페어를 이루어 더하는 대신 짝수번째의 값만 더해주는 것이다. 이는 어차피 min()값이 
짝수번째의 값이 될 것이기 때문에 가능한 방법이다.

```python{numberLines: true}
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        sum = 0
        nums.sort()

        for i, n in enumerate(nums):
            # 짝수 번째 값의 합 계산
            if i % 2 == 0:
                sum += n

        return sum
```

파이썬을 사용하여 위의 방법을 정리한 것이다.

```python{numberLines: true}
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])
```
